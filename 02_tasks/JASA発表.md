# 配分依存

歴史的解釈：
世界が複雑化し、天地と直接に代謝して資源を得る余裕のない人間が増えると、人工性の開発や管理を担う成員が生まれ、配分を通じて生存を維持するようになる。

その結果、資源の質と量を左右する〈配分枠〉――「何が価値ある成果か」「どんな人が評価されるか」を決める型やプロセス――に適応し、自らの性質を枠に合わせて刈り込むことを余儀なくされる。小原が指摘した〈自然さ〉の喪失や、社会理論で言う疎外（斎藤 2019）の多くは、この枠への過剰適応、あるいは適応不良に起因する（楊 2023；Yang 2023；Yang 2024）。

この包摂は、支配や抑圧に完全に還元できない。
楊（2021）では、進学競争への外部的強制がないオルタナティブ学校においても、生徒が自発的に「より良い未来を確保するための資源」を求め、配分依存に組み込まれていく過程が確認された。配分依存は外部権力の押し付けだけで成立するのではなく、資源獲得の論理への自発的参入としても展開する。

# 貨幣依存

古砂漠の民族（Graeber 2011）や農業税基が足りなかった宋朝（Von Glahn 2016）、イギリスのエンクロージャーで農地から追い出された元農奴（Marx 1976；Polanyi 1944；柄谷 2010）など、天地依存でも配分依存でも生存資源を十分に確保できない場合、第三の依存媒介が必要とされる。

これは人格的関係や天候といった不安定要因に左右されない、「社会に対する請求権」によって資源獲得を可能にするシステム、すなわち「特定の目的から解放された最も純粋な道具・手段」としての貨幣である（Simmel 1990）。これによって地理的空間や配分枠からの解放が可能となり、貨幣は人間を強く吸引する誘因ともなる。

配分依存が〈配分枠〉への同化によって資源を得るのに対し、貨幣依存は、購買力をもつ者の「満たされない需要や欲望」に応じて交易が成立することに基づく。

購買力があっても未満足需要・欲望が乏しければ交易は成長しにくく（Keynes 1936）、もし全ての人が心身ともに満たされた健全状態に達すれば、交易は動機を失い、貨幣依存システムは維持できなくなる（Baudrillard 1998）。

一方で、購買力が不足していれば、商品やサービスの購入によって需要や欲望を満たすことは困難となる（Galbraith 1958）。

大衆の購買力が需要や欲望を満たしきれない水準にとどまること、そして需要や欲望を非貨幣的に満たす手段が不足することにより、未満足需要・欲望が購買を焚き付けつつ、その前提としての貨幣稼ぎを強制し、生産と交易を継続させる（Graeber 2011；Marx 1976）。

こうして貨幣依存システムは、人間の欲求が恒常的に充足されないことを前提に自らを再生産し（Polanyi 1944）、購買力の格差を通じて貨幣流通を維持するという内在的矛盾を抱える。

# 制御性依存

人工性は多様だが、近代化以降、世界的に支配的になった人工性がある（楊2024b；Yang 2026）。それは、不確実性を減らし、〈意のままにならない〉状態を小さくする方向に社会を組み替えていく力である（上柿 2021）。これを〈制御性依存〉と呼ぶ。

制御性依存とは、資源の不安定さや脆弱さ（Rosa 2020）を下げるための制御手段によって、生活を成り立たせる依存の様式である（Yang 2026）。

ただし、人間があらゆる物事を制御手段に委ね、〈意のままになる生〉（上柿 2021）を追い求めるほど、世界の運行における人間の役割は小さくなり、制御手段は自動化していく。

小原先生がおっしゃる「カプセル」です。
フランクフルト学派、特にアルドノの「第二の自然」。

人工性が人間の関与なしに自足的に機能するようになり、人間がそれに合わせて生きる側に回るとき、その人工性は生存手段を超えて、新しい背景環境として立ち上がる。これを本稿では偽境(メタバース)と呼ぶ。

小原・羽仁（1995）が警告したように、人工環境が〈カプセル化〉するほど、人間はその内部への適応を強め、野外（天地）で生きる力を失っていく。配分依存や貨幣依存、制御性依存が強まるほど、当事者は天地との直接的な代謝から切り離され、資源循環の手触りが見えにくくなる。その結果、電力・物流・通信などの依存媒介が一度でも詰まれば、生活は連鎖的に脆くなる。

本来は宇宙への適応手段であるはずだった〈カプセル〉そのものが人間の制御から独立し、宇宙に還元できない固有の作動原理を持つようになると、人間が適応を強いられる新たな背景環境として立ち現れる。これが本稿でいう偽境である。

さらに、宇宙がすでに人間の介入によって攪乱され、暴走してしまう状態と重なると、偽境の独立は、人間の宇宙への集団的適応手段を失わせるだけでなく、人間に偽境そのものへの適応をも強いる。こうした人間条件とは、宇宙と偽境という二重の背景環境の狭間に置かれ、相反する二重の生存プレッシャーに同時に圧搾される境地である（Yang 2026）。

人工性は人間を助けるために発達するが、制御性が自足化すると、人間は「助けられる側」から「適応させられる側」へ転じうる。SF的な終末像に限らず、宇宙と偽境の二重圧搾の中で生存手段を失う環境適応的な窮地も、人類絶滅の生態学的な一つのシナリオとして捉えられる。
