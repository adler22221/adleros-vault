# [[張潔平]]

香港如同中國與俄羅斯等極權社會，被逼迫到不得面對「作為公共品的資訊」如何存活的問題，因此有非常多人投入Web 3。但台灣目前不處於這個狀態。因此，如何將更普世的經驗帶給所謂的「正常社會」是張潔平本次分享所探討的問題。

她試著從媒體、出版，或廣義的「資訊傳播業」所共同經驗的「焦慮感來自哪裡」為切入點。

若說資訊傳播業是座建設中的巨塔，不論是結構最完整也最難動搖的底基「學院教育體系（台大新聞所）」，或依序往上的「傳統出版社（聯經）」「正規網路新聞媒體業者（端傳媒）」「零工經濟下的內容生態與寫作社群（在場計畫）」與「型態仍在生成中的複合式傳媒（Matters Lab與飛地書店）」，張潔平先後或同時身處著這些位置，親身體驗大家各自的困境。角色換得多，也漸漸形成整體性的思路，有了更清楚的問題意識。

她認為傳媒跟學院一樣，是個封閉產業。大家焦慮的問題，如「網紅崛起」「記者養不活自己」「產業崩塌」等等，其實都只是看到「現象」。透過不斷換位思考，遊走於各種可溝通與不可溝通的藩籬間，她慢慢理解現象背後，是種新時代的結構：

「網路狀態下的知識，不是靜態的孤島或尖塔，是無邊無際的巨流河，可以不斷更新、不斷修正、不斷進化；也更符合知識的原貌。」

不論是學院或傳統媒體，都曾扮演「壟斷的資料庫」，被大眾強烈依賴。然而，資訊的壟斷逐漸崩解，大眾不再能得到固定的答案，世界之於大眾也顯得更加混沌。

「當知識變得網絡化之後，房間裡最聰明的那個人，已經不是站在屋子前頭給我們上課的老師，也不是房間理所有人的群體智慧。房間裡最聰明的人，是房間本身：是容納了其中所有的人與思想，並把他們與外界相連的這個網。」她引用哈佛大學Berkman Klein中心教授David Weinberger的話，描述了這個新時代知識與智能的特質。

人們被時代扔進了空前的不確定與不可預測性之中。出於不安或欲望，人們依賴機器，試圖重新控制一切。於是演算法成了最聰明的東西，很可能會反噬我們。

「資訊極度豐富，演算法令視野空前狹窄，如何克服一種知識盡在掌握中的自大感？」

張潔平認為，「群體智慧」是人類在不被演算法反噬的情況下，擁有回應不確定與不可預測性的關鍵。

「網路創造了深不可測的複雜性，超出人類的理解能力；同時更低門檻的、無限的可能性。混沌狀態下，試錯成為更常見的判斷策略，不是縮小可能性，而是去儘可能創造更多的可能性。但這跟我們的教育之間有明顯的張力。」

比起學院式的教育，她認為形成群體智慧的核心是「社群」——或用漢娜・鄂蘭的話說，就是得「坐下來對話」。

互連網從Web 1邁向Web 2時，早已讓我們見證群體智慧的力量。當時，許多人都想將百科搬到互連網。如nupedia允許專家自由但免費地撰寫百科條目，encarta則是透過付稿費作為專家撰寫條目的動機。然而，只有由「任何人」共同撰寫的維基百科存活了下來。這告訴我們：開放協作，可以帶來知識繁榮。

當互連網從Web 2邁向Web 3，則彷彿展開「土地改革」。不論是Web 1或Web 2，都沒碰到「產權」的問題。若說舊互聯網是整片領海或公海，Web 3時代的互連網則是每一滴水都標記著個人所有權的大海。

然而，基於「私有」的「共享」，恐怕也潛藏危機。她不禁問：「新聞/出版真的可以是讀者的Uber和記者作家的airbnb嗎？教育/課程真的可以是學生的Uber和學者的airbnb嗎？」

內容的「公共品」性質，不能單靠市場機制解決。Matters一開始雖然是往「公共」的方向走，但做著做著就懷疑了，因為這樣運作不下去。稍微往「私有」「個體化」的方向轉，雖然可以釋放更多價值，但也帶來更多問題：

「一個作者從選題到起心動念，再到為寫作/記錄去積累田野素材、做採訪研究，到呈現出文本，編輯修改，到發表至有影響力的平台讓更多人看見，進入公共領域，引起關注、解讀與辯論。這裡每個環節都需要支持。當新聞產業變成獨立記者/自由撰稿人們的零工經濟，怎樣構建一個市場可持續、規模可擴展的支持網絡，給公共討論提供有品質的基礎？」

若說AI是右派的極致，區塊鏈則彷彿左派的極致。在徹底資本主義化與徹底社會主義化之間，「如何組織公共社會來創造公共品」，才是去中心世界真正要解決的問題。針對這個問題，張潔平暫時的思路仍是「社群」：不追求聚集一大群人的力量，而是走一條接近小國寡民、小農市集的社群主義之路。

正如Adobe的CPO暨Behance創辦人Scott Belsky所說：預測科技的未來時，不能忽略人們發自本性對「過去」的留戀，以及復歸於親近感、關係性與村落型態的追求。
