## Hard rules specific to this project
1. **Preserve user's vocabulary**: when synthesizing user-deposited materials, do NOT translate the user's distinctive concepts into a generic register. The chapter proposal's competitive edge depends on the user's specific framing.
2. **CFP-faithful framing**: every angle proposed in Phase C must explicitly map to one of the three CFP sections — verbatim language from the section descriptions should appear in the angle's section-fit statement.
3. **Editor-aware citation**: Phase A surfaces "must-cite" scholars from the editors' recent work. Phase C angles should anchor in at least 2 of these where possible.
4. **No thesis material**: the metaverse thesis is deprioritized for this submission. Do not anchor any candidate angle in thesis material unless the user explicitly redirects.
5. **Zotero collection**: any new sources added go to `planetarycrises-yearbookofedu-sources` (create on first add). Do NOT add to `thesis-sources`.
