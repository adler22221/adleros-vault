# 逍遙的條件：無待

1. 對待：彼此、物我、
對比，兩兩相對的意思，相對的概念

哲學：
- absolute --> 無待，沒有彼此、物我、貧賤、高下、美醜，超越相對之外 
- relative --> 有待

在基督教也是
人間有沒有善惡？--> 相對的
但上帝是善的嗎？--> 絕對的

俗諦
真諦

從超越角度看世俗爭論，所以平齊是非

2. 依待：condition
成為可能，也成為束縛
無依待：無條件，自由自在

3. 對待跟依待的關係？

根本上應該是去了對待才能減少依待
平齊富貴貧賤沒有分別意義
都無所謂了
不會因此哀樂得失
這種情況自然不會有所依待

一定要正價值，不要負價值，就很有條件
從道的角度，從超越的角度
在世間成就的或許少，但束縛牽絆少
就自然逍遙

所以逍遙遊講完一定要講齊物論
因為齊物論才是本
把對待減少，依待自然變少，就有機會進入逍遙之境

掌握的這個主旨就至少掌握了一半

逍遙遊是境界
要到達境界需要功夫
功夫：「至人無己，神人無功，聖人無名」這些都是對待產生的依待
根本在無己，才能無功無名

方勇是老實的學者
沒有太多才情豪氣
不像楊如賓

理解古典時
「以經解經」
不要看字典，因為可能無關
重點是看經文自己怎麼用

透過經點內部的文字說明瞭解文字用法

把大鵬當作理想標竿是搞錯了
都還是有待的

溟代表很遠
透過文字產生一種聯想
不像白話文直白沒有想像空間
鯤是什麼？不要被枝節分心

海運則將徙於南溟--> 大鵬要等時機 
「怒」而飛 --> 需要努力，對比小鳥隨時可飛隨時降落（決而飛）


搏？摶？
搏：拍打

息：風、休息
六月息：六月的風 vs 累積六個月的風才能往南飛（朱老師解）
三千里、九萬里、六月息 --> 都是誇張的詞


對道家來講堯舜是大鵬
依舊有待
