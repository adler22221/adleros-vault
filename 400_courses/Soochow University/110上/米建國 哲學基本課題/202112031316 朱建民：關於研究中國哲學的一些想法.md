## 字詞意義的二分：內涵 intension、外延 extension

外延：所指涉的對象
內涵：意思

中華民國總統、三軍統帥
- 外延相同 --> 指涉同一個人
- 內涵不同

## 意義二分只是為了方便討論而採取的基礎

例如，意義可以有三分
- 借用福雷格（[[Frege, Gottlob]]）"on sense and reference"的用語
	- sense -->  intension 內涵
	- reference -->  extension, denotation 外延
	- color --> connotation, implication 言外之意
言外之意在自然語言中有重大作用，但不夠客觀，不夠明確
- 聽不懂的笑話、體會不出的詩詞之美
- 言者無心，聽者有意
- 狹帶情緒反應：e.g. 納粹一詞的情緒固化後，即轉成言內之意 --> 已經帶有顏色 --> 顏色固定，意思跟著固定
	- 台灣高中生惡搞納粹手勢：手勢本身沒有什麼意涵，但德國文化交流中心馬上提出抗議
--> 意思遠遠超出內涵與外延

## 文學歡迎言外之意，哲學未必
對文字的敏感度、豐富的想像力，可以讀出許多言外之意
但對追求精確的（當代）哲學討論，並無裨益
--> 此間基礎不同，難言優劣

自然語言的意義側面，不止於三分，還有可能係分下去
但為求討論有共同基礎，一班還是回到較具客觀性的二分
以簡化架構看世界，看久了，莫以為世界僅止於此
--> 哲學其實帶著簡化架構 --> 不要被自己的架構騙了
簡化架構 ≈ 成見，反映時代氛圍
--> 承認觀點的有限性，不要太多排他傾向

「中國哲學」的意義＝「中國哲學」的內涵與外延
避免涉及「中國哲學」一詞的言外之意或附加情感

# 一般使用的定義法

外延定義法 extensional definitions
- 實指法/直示法 ostensive definition (demonstrative definition, definition by pointing)
- 列舉法 enumerative definition (definition by example)
--> 這只是

內涵定義法 intensional definitions
- 必要條件法 necessary and sufficient conditions
	- 種差法 (genus and differentia，雅里士多德常用) 人為理性動物 --> 社會動物是必要條件，但不是充分條件/不是differentia

最完備的定義即是把所有的必要條件全部列出 --> 充分條件
通常得退而求其次，條件不充分，列舉不全盡，以下即是

# 定義背後的價值選擇

條件不充分、列舉不全盡，幹嘛呢？
--> 背後有選擇，有關注，有價值預設

很難遇到完美定義（不容易，有時也沒必要），通常是不充分、不全盡，標舉出來的必要條件乃是被挑選出來的（selective）
挑選及代表關注重點，可能涉及價值偏重
--> 因為需要、因為關心所以挑出來
主動挑出來 --> 反映價值取向

雅里斯多德定義人為理性的動物
孟子：人之異（differentia）於禽獸（genus）者幾希（essence）。人之所以為人者。無惻隱之心，非人也（必要條件：modus tollens）。
--> 兩種角度作出的結論
--> 這麼多可以定義人的方式，為什麼這樣定義？ --> 這就是哲學，一定涉及價值取捨

1960年學術強調價值中立 --> 曾經受束縛的主張，希望擺脫
但怎麼可能擺脫？
重點是不無條件接受信仰某種價值
但不否定經過理性、慎思明辨後的價值

# 通過對比看「哲學」的特性

生存的回應：
不論是原始或最先進的科技 --> 為了生存存活，以生物需求為基礎，做生存回應
--> 但哲學超乎生存需求的滿足與提升，跟生存關係不大 --> 「無用之用是為大用」的自我安慰，沒有實用價值

感性的回應：
不能沒有感性的回應 
--> 如果哲學家對世界沒有感性的感受，或許可以做哲學，但至少不能做中國哲學
一般的哲學家對世界、人生有極為強烈深刻的感受

文學家、藝術家用豐富的方式來「展現」自我存在的感受
但哲學不停留在「展現」，而要持續「追問」
托爾斯泰《伊凡伊利奇之死》（沒有）vs卡繆《西西弗神話》（開始有哲學）
賈選凝：羅卓瑤的影像裡，住著唐君毅的靈魂
一般人會鎖定在時代、港台歷史脈絡
但哲學怎麼可能鎖定在特殊脈絡？哲學是超越的
羅卓瑤的感動跟時代革命是不一樣的
但羅卓瑤是感性的回應，還是跟唐君毅不同

信仰的回應：
不以信仰為起點
（雖然經過審思選擇信仰）

理性的回應：
科學追尋原因（cause）、解釋系統
哲學找尋理由（reason）、目的、意義、價值（中西折）
目標不同，有其擅長方法
哲學方法：分析、批判、體證（古今中外各有偏重）
--> 根源所以可能的依據
--> 體證，實質的體悟，實踐，學而知之
（科學不太會搞基本概念、批判定義）

古今、中外有別：
有的偏體證、批判、分析

# 當代規範的實質層面

何謂學術：
經過證明的新知識，以適切的文獻紀錄作為根據 properly documented proven new knowledge
透過「眾人」（--> 不期待英雄、智者、先知）「長期」的努力，不斷「豐富」（學術只收原件不收複本）人類的「知識寶庫」

預設一定的知識觀：知識是經過證成的真信念
--> 未必是傳統東方最重視的，未必中世紀西方所最重視的

進入學術寶庫的知識必須是新的
只收藏原件，不收藏複本
「擴大」強調創新而非重複
旨在「擴大」--> 既有知識的學習與推廣=教育，非學術
整理、彙編、翻譯...都不算

創新在博士論文的評比標準

擴大：預設知識屬於可累積的
不可累積的不在其內

張載：見聞知vs德性知
德性知不可累積 --> 不在學術範圍內
連雅里斯多德都不算

程明道以不麻木說仁
但原初之「深切體悟」不屬於可累積的

獲得「不麻木說仁」的知識不難
但能不能獲得「體悟」
程明道說這句話只是指引，不是最終目標
目標是「體悟」
體悟不必然超過程名道 --> 可見不可累積性

## 當成活的哲學會對研究者產生影響

不只是把它當作道理、邏輯論證來看
而是有所感受、實際在「溝通」

幾年前中國請Michael Sandel來開哲學會議
最驚訝的是安樂哲、黃勇

[[安樂哲]]：
美國人，先到香港跟勞思光學習，後來到台灣跟方東美學
後來成為知名漢學家
可以用中文教中哲，但更深入的還是沒辦法只能用英文

黃勇
在美國教中哲
用英文教中哲

為了配合michael sandel用英文

安樂哲講中哲，充滿感情，形式也非常中哲
很像方東美、唐君毅很類似

黃勇完全是西方分析哲學的方式
中國臉孔，全是西方思維

時空錯置？

--> 重點不在國籍，而在態度抉擇

黃勇：用西方的方式一爭高下，沒有產生中國哲學式的影響，反而是分析哲學式的影響
安樂哲：喜歡中哲，道理產生認同感，對人生觀、形式風格產生影響

為何不適合完全用外人角度、客觀、價值中立地去看中哲？
玻璃外看玻璃內，只看到動作，聽不到聲音，感受不到溫度，會把太多東西過濾掉
--> 中哲的精彩，來自生命的「學問」，而不再概念的清晰，分析的細密，論證的嚴謹（後者是西哲的精彩）
真要比概念、分析、論證 --> 去看天台宗的智者大師，但不是主流
生命的學問主要在回應生命大問題、根本問題

曾研究唯識宗：經典 中文、梵文、英文、各國語言參看
講半小時，但再問就講不下去了 --> 忘了唯識宗的本質，只看到皮毛，只看到「唯識經」
缺了太多東西

馮友蘭認為良知是假設，熊十力直斥之為而謂良知乃是一種呈現
「假設」是西方的看法，隔著玻璃看
若以良知為假設，順此而下者皆不相應 --> 基礎就不對了

「己所不欲勿施於人」用邏輯檢視內部矛盾，但不管怎麼寫，都是不相應的
不是真假語句、陳述語句
而是指引實踐方向的語句
應以西方狹義的西方知識觀，不只看不出中國哲學的精彩，甚至曲解了中哲 --> 不是不能講，但是不相干

生命的學問必須關聯於生命的體會
未受毀譽糾纏之苦，難以體會莊子逍遙遊「舉世而譽之而不加勸...」
