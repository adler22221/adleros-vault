# 彭揚凱

居住與房地產政策分開談

房價指數

不夠有錢 → 不可能不用擔心居住
租屋黑市 → 住不安穩
不夠窮 → 無法取得住宅補貼

過去邏輯：
- 補救邏輯
- 市場邏輯

新邏輯：社會投資
目標：居住安定感

20-30不可能買得起→擴大健全租屋市場
30-40→中間型居住選擇

重新打造住房階梯

社會投資：
1. 住房補貼
2. 健全租屋市場
3. 中間型住宅
規範管理
4. 購屋

台灣特殊性：
租屋市場小：沒有精確數據12-13%（國外3-5成）
租屋黑市：所有租屋行為都不在政府規範內，沒有品質規範，逃漏稅等等
- 不可領補助 
- 不可設籍→影響政治與社福權益
- 租金抵稅用不到

如果多數人都違法，代表法形同空設 → 租稅大赦
提高租金抵扣額（錢根本沒進口袋→特別扣除額）
高空屋率高房價，持有成本低→囤房特別稅 提升供給
軍公教房東：租金課稅，級距會翻升
→ 分離課稅
→ 稅率很低

囤房稅出租免徵稅
