### 内容説明

本書は、近代社会の歴史を階層システムの変容としてとらえるという視点を提示している。これは、「階級」という狭く限定された概念枠組みや、「構造」という固定されたイメージを超えて、階層的諸現象が政治、経済、教育、家族、社会意識などと複雑に絡み合って社会のしくみを構成している。そのしくみが近代において変容を遂げてきた。そして、さらに変容しつつあることを考察している。

## 内容紹介

21世紀の市民社会における階層システムはどうあるべきか．平等＝正義，不平等＝悪というイデオロギーを超えて，公平原理と公共的価値の必要性を説く．さらに政治・教育の課題，社会的弱者の問題，グローバル時代の市民権など，今後の階層研究のあり方を提起する．

# 2 公平の評価基準(p.9)

|      | 平等 | 公平 |
| ---- | ---- | ---- |
| 平等 |      |      |
| 公平     |      |      |

> 平等は公平の一つの下位基準である。

\- p.10

公平の四つの下位基準：
- 平等
- 必要
- 努力
- 実績

> というシリーズ記事のなかに、「公平な不平等」は良いけれども「不公平な平等」は困るといった趣旨の内容が掲載された。「公平な不平等」と「不公平な平等」という表現を対比的に使っているということは、平等と公平が二つの独立した概念として使われていることを示している。いま、二つの次元をクロスさせて表示してみよう。公平と平等の二つの軸の組み合わせから四つのセルができる。世間で容認されているのは、セルの1と2である。容認されていないのはセルの3と4である。マスコミをはじめ多くの論調が、公平さを求めていて、平等それ自体を追求することは少なくなったようである。

|      | 平等 | 不平等 |
| ---- | ---- | ---- |
| 公平 |   1   |   2   |
| 不公平     |   3   |   4   |

> 従来は、平等かどうかの一本槍で社会を評価していたのが、いまでは新しい次元が付け加わったのである。後章でもとりあげるように、最近になって==保護政策や補助金政策は「社会的弱者」一般ではなくて、その中の「真の弱者」に対して行うべき==であることが経済学の分野からではあるが指摘されるようになった(八田·八代1995)。このことは、先の「不公平な平等」は困る、という言い方と呼応している。単なる平等化動機とは異なる。なぜならば、「真の弱者」は「社会的弱者」のなかの部分集合として存在しているはずであり、したがって、「真の弱者」がそれ以前には社会的参加の機会から排除されていたとか、十分な資源配分に与かっていなかったというのとは訳がちがうからである。

> それぞれのセルが日本社会ではどのような状況を反映しているかについては、別稿(高坂2000a)に委ねるとして、ここではもっとも議論の多いセル3についてのみ取り上げることにしたい。〈セル3:平等ー不公平〉
> 
> 平等が実現しているのに、なぜ人々は不公平と受け止めるのか。
> 
> 戦後の日本社会では「平等」価値が大切にされた。その結果、平等価値の実現のためには、弱者保護を行わなければならないという認識につながった。農業部門、中小企業、高齢者、地方などは保護すべき弱者というカテゴリーを構成してきた。ところが、==弱者保護政策の結末は、「弱者」の画一的解釈^[as long as there is a need for [[Allocation Dependence 配分依存]], wouldn't this always be the case, just a matter of degree?]とその結果としての財政負担の増大であった。==最近よく言われるように、わが国における「弱者」の実体の一端は、官僚と企業の癒着構造の中での「利権の保護」や既得権益の保護になってしまっている。私たちにとっては、真に必要な弱者保護といわば「誤った温情」との区別をつけることが必要である。
> 
> ==政策ターゲットを絞り込むための有効な指標に辿り着くことができないまま、いわばカテゴリーミスが起こった。==すなわち、保護ないしは優遇すべき存在だと==認定された層のなかに実際には階層分化が見られ、その「階層内階層」を無視して優遇策を講ずることは、「悪平等」になる。==^[部分呼應[[202202221313 關於一雄欣欣中風的分配依賴體悟]]]「社会的弱者」保護を目的とした政策であっても、平等化志向が加わると、結果として不公平になってしまう。
> 
> ==平等にするために補助金をはじめさまざまの優遇措置を講ずる結果、マクロ的に見れば、当該社会の財政負担が増大してしまっているということが起きかねない。==^[The socio-political consequence of [[Allocation Dependence 配分依存]]]ちょうど、七○年代のイギリスの「福祉国家」政策がその放縦によって「イギリスの衰退」を招いたようにである。じじつ、これが起こったことである。少子超高齢化社会の到来は、国家に未曾有の負荷をかけることとなった。


> 平等原理はしばしば他の資源配分原理、たとえば能力主義や実績主義とぶつかる。^[so there is indeed a general relationship between fairness and [[Allocation Dependence 配分依存]]]したがって力や実績が認められないとか、能力や実績の面で劣る人に対しても平等に資源が配分されていくのだとすると、やる気をなくす人もでてくる。そうすると社会全体としては活気を失っていくことにもなりかねない。先にも述べたように、人々は今の時点で、必要や平等よりは実績や努力を選好する傾向のあることがはっきりした。つまり、人々は平等であることについては納得していないように思われる。

## 3新しい市民社会像に向けて

新しい市民社会は、==資源配分の観点から見る限り、公平をメタ原理ないしは統合的な配分原理とする社会である、と主張したい。==公平は上に述べたように、さらに四つの資源配分規準からなる。《平等〉は四つの公平規準の一つである。〈平等〉は、結果の平等を示している。それに対して、〈実績〉と〈努力〉とは現実には帰結として資源配分上の不平等をもたらすだろう。したがって、公平な社会は、構成原理として平等を志向する部分と、不平等を当然の帰結として容認する部分の二つから成る。〈必要〉は「底上げ」の議論であって、単純に平等を志向するとか不平等を志向するとかは言えない。

重要なことは、争点の内容と資源の配分状況とによって発動すべき規準が異なってくることであるでは、どのような争点のとき、どのような資源配分状況にあるとき、どの規準が発動されるべきか。

必要：
- 「健康で文化的な最低限程度の生活の営む権利」 --> 最低限度とはどう定義する？

> また、憲法の第二五条は「生存権、国の社会的使命」となっており、自治体の責任に傾斜した「シビル·ミニマム」という考え方とはずれるようにも思われるが、ここでは責任主体が誰であるかの問題には立ち入らない。さらに、「最低限度の生活を保障する」ことを目的に掲げている生活保護法の考え方も、〈必要〉に立っているけれども、何が最低限度であるかの特定は現実には難しい。なぜならば、必要の認識、とくに自分(たち)にとっての必要は自分のアスピレーション.レベルとの相関で決まってくるからだ。最近、鳥越(1997)は人間にとって生存権よりは生活権の方が必要だという主張を繰り返している。そうした視点を導入するならば、ここでいう〈必要〉も、決して生存してゆくうえでの必要最低限の資源配分を保障するということではなくて、人間らしい幸福な生活を送るのに必要な資源配分を保障するということでなくてはならない。

実績：

> 〈実績〉と《努力〉については、結論から言えば、上に述べた〈平等〉や〈必要〉を適用すべきであるといった争点や領域以外のところに用いられるべき規準である。〈実績〉原理は賃金体系のなかでは「成果主義」として定着しつつある考え方である。==実績を評価するのは誰か==、実績を評価する場面は国家レベルの話なのか組織レベルの話なのか、などはここでは問題ではない。一般的規準がここでの問題である。同じことは〈努力〉についても言える。努力の評価主体が誰か、努力は何に向けての努力なのかなどはすべて括弧に入っている。また、〈実績〉も〈努力〉も、==どの範囲の空間や時間に照らして評価されるべきか==という問題も重要な問題として残されている。たとえば、何かに照らして一年毎に実績を評価しようとする場合と、10年に一度評価しようとする場合とでは評価の内容も大きく変わってくるかもしれない。このことは〈努力〉についても当てはまる。
> 
> したがって、〈実績〉にせよ〈努力》にせよ、原理の実際の運用にあたっては評価対象とする時間的な幅や空間的な広がりについて明確な規準を設けていく必要があるだろう。〈実績〉も〈努力〉も、ともに==人の行為のポジティヴな側面を評価しようとしている==点でも共通している。〈実績〉のもとにおいては、実績をあげた者はそれだけ応分の資源配分を受ける。〈努力〉のもとにおいては、努力を重ねた者はそれだけ応分の資源配分を受ける。努力の結果、実績があがったかどうかは問わない。こういう風に言うと、==これらの原理はともに加算的(人のすぐれた側面ばかり認めようとする)規準のように思えてくる。==しかし、そうではない。〈実績〉にしろ〈努力〉にしろ==それらが不足しているものには資源が配分されないことを含意している。==実績があがらなければ、とことん、配分されなくなる。努力をしなければ、当然資源の配分は得られない。その意味では、ともに==減算的であり「きびしい」側面と緊張を含んでいる==。
>
>〈実績〉と〈努力》がたがいに独立の判断規準を構成することについては、努力をしても実績のあがらないこともあれば、努力せずに実績をあげることもあることを思えば、納得できるだろう。上には二つの原理に共通した特徴を述べてきたが、視点の持ち方の点で根本的に異なる点もないわけではない。〈実績〉の評価基準は個人の視点ではなく、集合体(組織、地域、国家、国際社会など、具体的にはさまざまだが)の視点にもとづく。組織には組織に対する貢献の見地から人事考課がおこなわれ、国家は国家なりの顕彰システムをもっているように、==〈実績〉という場合、集合体からみた判断規準が存在していることが前提になっている==。
>
>他方、〈努力〉は個人の視点である。学生の成績評価においては、単にリポートや試験における成績ばかりでなく「出席点」を加味することが少なくないが、このときの「出席点」は一種の努力の指標として扱われているに他ならない。集合体にとってこれといった貢献をしない人間にも資源配分の上で考慮しようという発想は、裏返しとして、努力の欠落に対する負の制裁という発想を伴っているが、この発想が現代日本において六割近くの人々に支持されているかに見える背景には、戦後日本があまりにもゆとりのないかたちで業績を人々に求めすぎたことへの反動や反省があるのかもしれない(日本人の「努力好き」の社会的文化的背景については、本シリーズ2巻6章の「日本人の不公平感は特殊か」を参照されたい)。
>
>いずれの規準を適用するにせよ、共通して配慮されるべき事柄というものもあるように思われる。一つは、公平規準に関する成員間の合意形成である。問題になっている争点や状況においてどの規準を採択すべきかについては合意が必要である。現代は価値観の多様化の時代と言われるけれども、==判断規準が何かについて意見が分かれていたならば、どのような施策をとろうとも結果は成員間に不満が生まれ混乱を招くだけである==。^[as long as people are allocation dependent ([[Allocation Dependence 配分依存]]), the disagreement surrounding the allocation criteria (measures) will always exist]
>
>もう一つの配慮すべき事柄は、==いずれの規準を採るにせよ、社会全体としてはやはり「排除」ではなく「包含」が求められるであろう(Giddens1999)==。inclusion(包含)とは、不平等という静態的状態を指すのではなくて、不平等を生み出さないメカニズムを指す動態的概念である。inclusivesocietyとは、したがって一言で言えば、「落ちこぼれを出さない社会」(後藤邦夫の訳)である。
>
>たとえば、〈実績〉主義に徹すれば、優勝劣敗の仮借無き社会というイメージが伴うかもしれない。しかし、〈実績〉主義といえども採択規準に関する成員間の合意と「落ちこぼれを出さない」メカニズムが社会にビルトインされている必要があるのではないか、というのがここで言いたいことである。
>
>社会階層とは、当該階層のグループとしての意識や行動によってレベルの違いが存在するけれどもいずれにしても資源の不平等分配を伴っている(高坂2000b)。上に見てきたことは、いわば階層的秩序の正当性の根拠を探っていたことに他ならない。来るべき社会において、適材適所の規準が採用されるようになるかどうかは最終的には分からない。しかし、現時点では人々の間で理想とすべき判断基準にはいくぶん落差がある(阿部1998)。すなわち、==学歴が高いほど、実績による分配が望ましいとする人が増え、逆に学歴が低い人では、努力をあげる率が高い==。==所得階層について着目しても上層は実績、下層は努力という傾向が見られる==^[reflects different classes' proximity to different kinds of allocation criteria/measures ([[Allocation Dependence 配分依存]])]。こうした食い違いのもっている意味を吟味し、合意を形成していくことが当面の一つの課題となるだろう。
